Pig Latin Encryption

Requirements: Write a stand-alone Java GUI application called TxtCrypt. Use either AWT or swing. This application should allow anyone to convert any text into Pig Latin. Your application must provide a pair of TextArea (or JTextArea) components where the user can type in any text into the first, and see the encrypted results in the second. The input text is called the plaintext, and the resulting Pig Latin is called the ciphertext. You must provide the basic GUI and event handling. This must include a button, which when pressed displays the ciphertext for the currently displayed plaintext. (PigLatin class with encryption method provided.)
