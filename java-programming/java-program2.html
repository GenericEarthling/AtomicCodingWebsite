Temperature Conversion

Purpose: Java I class project

Requirements: Create a Java stand-alone GUI program that displays a Fahrenheit to Celsius temperature conversion table in an AWT Frame or a swing JFrame. The table should run from 0 degrees Fahrenheit to 250 degrees Fahrenheit, in steps of 10 degrees. Round the Celsius temperature values to the nearest integer. Display any temperatures below the freezing point of water in blue and any temperatures above the boiling point of water in red the rest in black.
